Perm Spoofer Setup Guide

Windows Setup

アカウント作成の手順を以下の動画で確認してください。

Clean Install

クリーンインストールの手順を以下に説明します。

  1. 上記のメディア作成ツールをダウンロードするには、「今すぐダウンロード」をクリックしてください。
  2. メディア作成ツールを開きます。
  3. 「別のPC用のインストールメディア（USBフラッシュドライブ、DVD、またはISOファイル）を作成する」を選択します。
  4. 「推奨オプションを使用する」を選択します。
  5. 「USBフラッシュドライブ」を選択します（最小8GB）。
  6. 使用するUSBフラッシュドライブを選択します（これにより、すべてのデータが削除されます）。
  7. インストールメディアの作成が完了したら、PCを再起動します。Windowsの起動中に「SHIFT」キーを押しながら「再起動」をクリックして、ブートメニューにアクセスします。

Disable Windows Defender

Windows Defenderを手動で無効にする

  1. スタートメニューを開き、「Windows セキュリティ」と入力してください。
  2. 検索結果からWindows セキュリティアプリを選択し、「ウイルスと脅威の防止」の「ウイルスと脅威の防止」設定で「設定の管理」を選択してください。
  3. リアルタイム保護をオフにしてください。

dControlでWindows Defenderを無効にする

  1. dControlはこちらからダウンロードしてください。
  2. ファイルを解凍してください。
  3. dControl.exeを実行してください。
  4. Windows Defenderを無効にしてください。

Disable Windows Updates

Windows Updateを無効にする手順を以下に説明します。

  1. Windows Update Blockerをこちらからダウンロードしてください。
  2. ダウンロードしたファイルを解凍します。
  3. Wub_x64.exeを実行します。
  4. Windows Updateを無効にして「今すぐ適用」をクリックします。

以下のイメージを参考にしてください。

Download Loader

ローダーのダウンロードとセットアップの手順を以下に説明します。

Visual C++ 再頒布可能ランタイムのインストール

  1. Visual C++ 再頒布可能ランタイム オールインワンをこちらからダウンロードしてください。
  2. ダウンロードしたファイルを解凍します。
  3. 管理者として「install_all.bat」を実行します。
  4. インストールが完了したら、PCを再起動します。

DirectX ランタイムのインストール

  1. DirectX ランタイムをこちらからダウンロードしてください。
  2. インストール時に「Bingバーをインストールする」のチェックを外してください。
  3. インストールを完了します。

ローダーのダウンロード

  1. ローダーはこちらからダウンロードしてください。
  2. ダウンロードしたファイルをデスクトップに保存します。
  3. フォルダを解凍し、.exeファイルをデスクトップに配置します。

※セットアップの詳細については、提供されたイメージを参照してください。

Spoof HWID

HWIDをスプーフィングする手順を以下に説明します。

  1. exeファイルを開きます。
  2. 「Please enter your license key」と表示されたら、受け取ったライセンスキーを入力してください。
  3. ログイン後、「1」を選択してください。
  4. 15秒ほど待ちます。
  5. シリアルナンバーが更新されたか確認してください。
